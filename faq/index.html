How do I solve quadratic equations?

Quadratic equations in the form $ax^2 + bx + c = 0$ can be solved using several methods:

  1. Quadratic Formula: $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$
  2. Factoring: Find two numbers that multiply to $ac$ and add to $b$
  3. Completing the Square: Transform to $(x+p)^2 = q$ form
  4. Graphing: Find x-intercepts of the parabola

Example: For $x^2 - 5x + 6 = 0$:

Using the quadratic formula: $x = \frac{5 \pm \sqrt{25-24}}{2} = \frac{5 \pm 1}{2}$

So $x = 3$ or $x = 2$

How do I solve systems of linear equations?

Systems of linear equations can be solved using three main methods:

  1. Substitution Method: Solve one equation for a variable, substitute into the other
  2. Elimination Method: Add or subtract equations to eliminate a variable
  3. Graphing Method: Find the intersection point of the lines

Example using elimination:

$\begin{cases} 2x + y = 7 \\ x - y = 2 \end{cases}$

Add equations: $3x = 9$, so $x = 3$

Substitute: $2(3) + y = 7$, so $y = 1$

Solution: $(3, 1)$

How do I factor polynomials?

Polynomial factoring follows a systematic approach:

  1. Factor out common factors first
  2. Identify special patterns:
    • Difference of squares: $a^2 - b^2 = (a+b)(a-b)$
    • Perfect square trinomial: $a^2 ¬± 2ab + b^2 = (a¬±b)^2$
  3. For quadratics $ax^2 + bx + c$: Find factors of $ac$ that add to $b$
  4. Use grouping for four terms

Example: Factor $x^2 - 9x + 20$

Find factors of 20 that add to -9: $-4 √ó -5 = 20$ and $-4 + (-5) = -9$

Answer: $(x - 4)(x - 5)$

What is a derivative and how do I find it?

A derivative represents the rate of change of a function. Geometrically, it's the slope of the tangent line at any point.

Basic differentiation rules:

  • Power Rule: $\frac{d}{dx}[x^n] = nx^{n-1}$
  • Constant Rule: $\frac{d}{dx}[c] = 0$
  • Sum Rule: $\frac{d}{dx}[f + g] = f' + g'$
  • Product Rule: $\frac{d}{dx}[fg] = f'g + fg'$
  • Chain Rule: $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$

Example: Find $\frac{d}{dx}[3x^2 + 2x - 5]$

$= 3 \cdot 2x^{2-1} + 2 \cdot 1x^{1-1} - 0 = 6x + 2$

How do I find the area of different shapes?

Common area formulas:

  • Rectangle: $A = length √ó width$
  • Triangle: $A = \frac{1}{2} √ó base √ó height$
  • Circle: $A = \pi r^2$
  • Parallelogram: $A = base √ó height$
  • Trapezoid: $A = \frac{1}{2}(b_1 + b_2) √ó height$

For triangles with all three sides known (Heron's formula):

$A = \sqrt{s(s-a)(s-b)(s-c)}$ where $s = \frac{a+b+c}{2}$

Example: Triangle with sides 3, 4, 5

$s = \frac{3+4+5}{2} = 6$

$A = \sqrt{6(6-3)(6-4)(6-5)} = \sqrt{6 √ó 3 √ó 2 √ó 1} = 6$

What is the Pythagorean theorem and when do I use it?

The Pythagorean theorem states: In a right triangle, $a^2 + b^2 = c^2$

Where $c$ is the hypotenuse (longest side) and $a$, $b$ are the legs.

Use it when:

  • You have a right triangle (90¬∞ angle)
  • You know two sides and need to find the third
  • You need to find distances in coordinate geometry
  • You want to check if a triangle is a right triangle

Example: Triangle with legs 6 and 8

$c^2 = 6^2 + 8^2 = 36 + 64 = 100$

$c = \sqrt{100} = 10$

üìä Statistics Questions

What's the difference between mean, median, and mode?

Three measures of central tendency:

  • Mean (Average): Sum of all values √∑ number of values
  • Median: Middle value when data is arranged in order
  • Mode: Most frequently occurring value

Example: Data set {2, 3, 3, 4, 5, 5, 5, 7}

  • Mean: $(2+3+3+4+5+5+5+7) √∑ 8 = 34 √∑ 8 = 4.25$
  • Median: Middle values are 4 and 5, so median = $(4+5) √∑ 2 = 4.5$
  • Mode: 5 (appears three times)

When to use each: Mean for normal distributions, median for skewed data, mode for categorical data.

What is the order of operations (PEMDAS)?

PEMDAS (Please Excuse My Dear Aunt Sally):

  1. Parentheses (or Brackets)
  2. Exponents (or Orders)
  3. Multiplication and Division (left to right)
  4. Addition and Subtraction (left to right)

Example: $2 + 3 √ó 4^2 - (6 √∑ 2)$

  1. Parentheses: $(6 √∑ 2) = 3$
  2. Exponents: $4^2 = 16$
  3. Multiplication: $3 √ó 16 = 48$
  4. Addition/Subtraction: $2 + 48 - 3 = 47$
