Problem 1: Linear Equation

Easy

Solve for x:

3x + 7 = 22

Solution:

To solve 3x + 7 = 22, we need to isolate x.

1 Subtract 7 from both sides:
3x + 7 - 7 = 22 - 7
3x = 15
2 Divide both sides by 3:
3x ÷ 3 = 15 ÷ 3
x = 5

Answer: x = 5

Check: 3(5) + 7 = 15 + 7 = 22 ✓

Problem 2: Quadratic Equation

Medium

Solve the quadratic equation:

x² - 7x + 12 = 0

Solution:

To solve x² - 7x + 12 = 0, we'll use factoring.

1 Find factors of 12 that add to -7:

We need two numbers that multiply to 12 and add to -7.

-3 × -4 = 12 and -3 + -4 = -7 ✓

2 Factor the expression:
x² - 7x + 12 = (x - 3)(x - 4)
3 Apply zero product property:
(x - 3)(x - 4) = 0

This means either (x - 3) = 0 or (x - 4) = 0

4 Solve each equation:
x - 3 = 0 → x = 3
x - 4 = 0 → x = 4

Answer: x = 3 or x = 4

Check: (3)² - 7(3) + 12 = 9 - 21 + 12 = 0 ✓

Check: (4)² - 7(4) + 12 = 16 - 28 + 12 = 0 ✓
