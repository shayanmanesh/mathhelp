The Quadratic Formula

For any quadratic equation $ax^2 + bx + c = 0$, the solutions are:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Understanding the Discriminant

The discriminant $D = b^2 - 4ac$ tells us about the nature of roots:

  • If $D > 0$: Two distinct real roots
  • If $D = 0$: One repeated real root
  • If $D < 0$: Two complex conjugate roots

Solved Examples

Example 1: Two Real Roots

Solve: $x^2 - 5x + 6 = 0$

Here $a = 1$, $b = -5$, $c = 6$

Discriminant: $D = (-5)^2 - 4(1)(6) = 25 - 24 = 1 > 0$

Using the formula: $x = \frac{5 \pm \sqrt{1}}{2} = \frac{5 \pm 1}{2}$

Solutions: $x_1 = 3$, $x_2 = 2$

Example 2: Complex Roots

Solve: $x^2 + 2x + 5 = 0$

Here $a = 1$, $b = 2$, $c = 5$

Discriminant: $D = 4 - 20 = -16 < 0$

Solutions: $x = \frac{-2 \pm \sqrt{-16}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i$

Other Methods to Solve Quadratics

1. Factoring Method

If $ax^2 + bx + c = a(x - r_1)(x - r_2)$, then roots are $r_1$ and $r_2$

2. Completing the Square

Transform $ax^2 + bx + c = 0$ to $(x + p)^2 = q$ form

3. Graphical Method

Find x-intercepts of the parabola $y = ax^2 + bx + c$
